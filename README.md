## Chat Games Lite

A lite Chat Games / Reactions plugin for Minecraft with configurable message, rewards etc.
Known to work on 1.20.4 (untested on newer versions)

### Built With

* Java

## Roadmap

- [ ] More configurability
- [ ] Add more chat games
    - [ ] Trivia Questions
    - [ ] Fill In The Blank
